Спроектировать структуру FILE, содержащую поля: каталог,
имя файла, расширение.
Спроектировать структуру RECORD, содержащую поле типа
FILE и дополнительно поля: дата и время создания, атрибуты
(«только чтение», «скрытый», «системный»).
Файл содержит информационные записи о файлах на жестком
диске. Требуется спроектировать класс, позволяющий обрабатывать
эти данные. В классе должно быть предусмотрено, как минимум,
выполнение следующих видов обработки:
1. Загрузка из файла;
2. Сохранение результатов обработки в файлах
3. Добавление записи;
4. Удаление записи;
5. Вывод на экран дисплея данных и результатов обработки;
6. Сортировка по атрибуту, а внутри атрибута по имени файла и
расширению;
7. Алфавитная сортировка по каталогу, имени файла и
расширению;
8. Числовая сортировка по дате и времени создания;
9. Перечень атрибутов с указанием числа файлов (результат
отсортирован по атрибутам);
10. Перечень атрибутов с указанием числа файлов (результат
отсортирован по числу файлов);
11. Поиск по дате и времени создания (результат отсортирован
по названию каталога, а внутри каталога по имени файла и
расширению);
12. Поиск по дате и времени создания (результат отсортирован
по атрибуту).
